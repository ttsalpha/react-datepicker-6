# [react-datepicker-6](https://www.npmjs.com/package/react-datepicker-6)

> This is a forked version of [react-modern-calendar-datepicker](https://github.com/Kiarash-Z/react-modern-calendar-datepicker) that makes the library compatible with react v17.

A modern, beautiful, customizable date picker for React.

## Installation

```bash
npm i react-datepicker-6
```

Or if you prefer Yarn:

```bash
yarn add react-datepicker-6
```
